Terminal Also Connects to Same Server

Parallel client File: client/client.py

  1. Terminal app connects to http://localhost:5000 using Socket.IO client.
  2. When terminal sends text, it emits send_message with sender Terminal.
  3. Server saves the message in MySQL, then broadcasts to all clients (browser + terminal).

Exact Message Sequence (Step by Step)

  1. User types in React UI or Terminal.
  2. Client emits send_message { sender, text }.
  3. server.py receives event and calls save_message(sender, text).
  4. dataStore.py inserts row into MySQL chat table.
  5. Server emits receive_message to everyone connected.
  6. All UIs append the message to chat window.
